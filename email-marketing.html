WePhoneShop 邮件营销方案

聚焦批发客户、潜在客户与欧洲分销商 · 固定节奏执行 · 以邮件带动月度批发订单 ≥10% 增长

一、总体目标（Objectives）

复购率 ↑
将一次性客户转化为长期合作商
品牌认知 ↑
树立"西班牙专业手机配件供应商"形象
≥10%
每月由邮件促销带来的额外批发订单
关系强化
定期发布新品/价目/库存/行业资讯
欧洲拓展
建立葡语/法语等欧洲语系客户库

二、客户分组与数据库策略（Segmentation）

🏪 B2B 批发客户

手机配件零售店、维修店、线上卖家

  • 邮件类型：批发价更新 / 新品目录 / 库存提醒 / 节日促销
  • 数据来源：订单、网站注册、WhatsApp

🧾 潜在客户

询价未下单、放弃购物车、展会名录

  • 邮件类型：欢迎邮件 / 产品目录 / 限时优惠
  • 数据来源：表单、名片、会后跟进

🌍 欧洲分销商

法国/意大利/葡萄牙等国配件批发商

  • 邮件类型：品牌介绍 / 分销政策 / 合作邀请
  • 数据来源：站外拓客、社媒、展会

平台建议：Brevo / MailerLite / HubSpot Starter · GDPR 合规（每封邮件必须有取消订阅链接）

三、邮件内容策略（Content Strategy）

1. 品牌类（Branding）提升信任

  • 主题示例：📦 WePhone – Tu proveedor de confianza en accesorios de móvil desde 2015
  • 要点：公司简介 / 仓储规模 / 15年经验 / 热销产品图
  • CTA：Descubre nuestro catálogo completo

2. 批发促销（B2B Promotion）拉动下单

  • 主题示例：💥 Ofertas por tiempo limitado – Hasta -25% en fundas y cargadores
  • 结构：折扣+库存数量 / 产品图+SKU
  • CTA：Ver precios ahora

3. 新品发布（New Arrivals）优先告知

  • 主题示例：🚀 Novedad: Carcasas magnéticas Puloka para iPhone 16 ya disponibles
  • 要点：功能亮点 / 兼容型号表 / 价目 PDF
  • CTA：Solicitar catálogo

4. 技术与行业资讯（Professional Tips）专业权威

  • 主题示例：🔧 Cómo elegir el cristal templado ideal para tu cliente
  • 目标：降低退货 / 建立专业度 / 关联推荐

5. 季节与节日营销（Seasonal）节点爆发

  • 节点：黑五 / 圣诞 / 开学季 / 夏季清仓
  • 节奏：预告（Preview）→ 正式（Live Offer）

四、自动化流程（Automation Flow）

主要自动化流程

流程详情

  • 🆕 欢迎邮件：新注册 / 加入数据库 → 品牌+价目PDF+首单券
  • 🛒 购物车挽回：放弃 24h → 提醒 + 额外 5% 优惠
  • 🔁 复购提醒：上次采购 45 天 → 新到货+热销推荐
  • 📦 库存通知：重新入库 → 自动邮件「已恢复库存」
  • 📅 每周汇总：每周一 → 新品目录 + 本周优惠 + 行业资讯

五、设计与模板规范（Design Guidelines）

🎨 视觉设计

  • 主色：#FF3B30（红）
  • 辅色：#1C1C1E（深灰）
  • 图片：产品白底 + 仓库实拍 + 统一水印

🧱 布局结构

  • 结构：Logo → Banner → 两栏产品 → CTA
  • 按钮：红底白字「Ver catálogo / Descargar precios」

🔠 字体与响应式

  • 标题：Montserrat Bold
  • 正文：Open Sans
  • 移动端优先：≥60% 邮件在手机查看

六、执行节奏（Campaign Calendar）

周期 内容方向 邮件类型 频率 主要目标
每周 新品上架 / 最新批发价表 新品更新 / 批发目录 每周 1 封 优先告知 & 维持活跃
每两周 批发促销 / 节日特价 促销 / 清仓 每两周 1 封 拉动下单 & 库存周转

十、阶段性实施计划（Roadmap）

阶段1 · 基础搭建（1–2个月）

  • 导入客户数据库，清洗与分组
  • 配置 Brevo 自动化流程
  • 制作 3 套邮件模板

阶段2 · 节奏化执行（3–5个月）

  • 每周新品、双周促销、月度报告
  • UTM 追踪 + KPI 看板复盘
  • 优化主题、折扣与CTA（A/B测试）

阶段3 · 优化与扩展（6–12个月）

  • 多语言分支（PT/FR）与分销频道
  • CRM 整合（购买记录 + 邮件行为）
  • 目标：邮件拉动月度订单 ≥10%

合规与备注

  • GDPR：每封邮件保留取消订阅链接；妥善管理退订与隐私请求。
  • 追踪：使用 UTM（如 ?utm_source=email&utm_campaign=weekly-new）。
  • 数据：本页面为视觉化方案展示，不含个人数据。
